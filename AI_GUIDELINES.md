# AI Development Guidelines

Este archivo define cómo las herramientas de IA deben trabajar dentro de este repositorio.

El objetivo es mantener consistencia, calidad técnica y evitar refactors innecesarios.

---

# Contexto del proyecto

Este repositorio contiene el código de la landing page del estudio **IT Integral Solutions**.

La arquitectura del proyecto es deliberadamente simple:

- HTML
- CSS
- JavaScript Vanilla

No se utilizan frameworks ni herramientas de build.

El objetivo es mantener el sitio **rápido, claro y fácil de mantener**.

---

# Principios de desarrollo

Cualquier modificación realizada por una IA debe respetar:

- código limpio
- separación de responsabilidades
- arquitectura simple
- evitar duplicación
- evitar dependencias innecesarias
